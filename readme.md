# 🧾 Garg Readymade & Hosiery Billing System

A **fully functional web-based billing and accounting system** designed for small businesses like garment shops and wholesalers. This application allows users to create invoices, manage suppliers, track purchases, maintain ledgers, and generate monthly financial summaries — all in one place.

---

## 🚀 Features

### 📝 Invoice Management
- Create and manage **sale invoices**
- Add multiple items with:
  - Description
  - HSN Code
  - Quantity & Rate
- Automatic calculation of:
  - Subtotal
  - Discount
  - GST (CGST/SGST/IGST)
  - Grand Total
- Convert total amount into **words**
- Print / Export invoice as PDF

---

### 🗂 Bill Tracking
- View all saved invoices
- Search by:
  - Invoice number
  - Customer name
- Delete individual or all bills

---

### 🏭 Supplier Management
- Add and manage suppliers
- Track:
  - Total purchases
  - Payments made
  - Outstanding balance
- Store supplier details:
  - GSTIN
  - Address
  - Contact info

---

### 🛒 Purchase Management
- Add purchase bills manually or via:
  - 📷 Image upload
  - 📄 PDF upload
- AI-based bill scanning (auto-fill fields)
- GST calculation support
- Payment tracking

---

### 📊 Ledger System
- Maintain complete financial records:
  - Sales
  - Purchases
  - Payments In/Out
- Real-time balance calculation
- Filter by transaction type
- Search functionality

---

### 📅 Monthly Analytics
- Monthly financial summaries
- Annual overview:
  - Sales
  - Purchases
  - Payments
  - Net balance
- Detailed month-wise breakdown

---

### 💾 Data Management
- Data stored in **LocalStorage**
- Backup system:
  - Export data as JSON
- Restore system:
  - Import saved data

---

## 🧑‍💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Storage:** Browser LocalStorage
- **UI Design:** Custom CSS (Responsive + Modern UI)
